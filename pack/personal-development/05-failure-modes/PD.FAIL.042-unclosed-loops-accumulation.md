# Антипаттерн: Накопление незакрытых петель (Unclosed Loops Accumulation)

## 1. Описание

Агент накапливает незавершённые обязательства: начатые планы, брошенные курсы, необработанные заметки, невыполненные обещания самому себе. Каждая незакрытая петля генерирует внутренний шум.

## 2. Симптомы

1. Ощущение «занят, но ничего не собирается»
2. Потеря доверия к себе («обещаю себе и не делаю»)
3. Заметки накапливаются без обработки
4. Множество полуначатых курсов/книг

## 3. Тест обнаружения

Посчитать открытые петли (начато, но не завершено и не закрыто осознанно). >5 необработанных → накопление активно.

## 4. Коррекция

Применять PRINC.032: каждый начатый цикл должен быть либо завершён до результата, либо осознанно закрыт с письменной рефлексией.
