# Антипаттерн: Притупление чувствительности к новому (Novelty Sensitivity Attenuation)

## 1. Описание

Со временем мозг агрессивно фильтрует новизну для снижения тревожности и энергозатрат: чувствительность к новой информации угасает («слушаю старые песни, сопротивляюсь новым технологиям»).

## 2. Последствия

- Резонанс становится статистически редким
- Творческий конвейер голодает без входного потока
- Мировоззрение застывает

## 3. Коррекция

- Систематическое медленное чтение (METHOD.003) восстанавливает чувствительность
- Echo-test протокол для распознавания слабых сигналов
- Понижение порога фиксации: ловить «маленькие» мысли, не только «великие идеи»
