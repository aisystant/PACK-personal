# [PD.FAIL.050] Self-Criticism as Resource Leak

---

## Definition

A pattern where the agent's inner critic operates continuously, consuming cognitive resources that should be directed toward creative and strategic work. The agent habitually criticizes themselves for mistakes, failures, and insufficient results — instead of formulating hypotheses for correction and moving forward. Self-criticism becomes a background process that drains attention without producing work products.

---

## Error Type

**Type**: `cognitive` — misallocation of cognitive resources due to habitual inner process

---

## Detection Test

| Symptom | Example |
|---------|---------|
| Post-error rumination | After a mistake, replays the event mentally for hours/days |
| Daily devaluation | "I didn't do anything special today" (despite completed work products) |
| Guilt without action | Feels bad about missed targets but does not reformulate approach |
| Invisible achievements | Cannot list 3 accomplishments of the current day/week |
| Critic as identity | "I'm my own harshest critic" (stated with pride) |
| Energy spent on regret | Time budget includes significant unpaid "self-punishment tax" |

**Test question**: After a mistake, does the agent formulate a hypothesis for non-repetition within minutes — or do they spend hours in self-reproach? If the latter, this failure mode is active.

---

## Why This Is an Error

| Self-criticism loop | Correction-and-praise pattern |
|---|---|
| Error → emotional self-punishment → energy loss → no behavioral change | Error → hypothesis → correction → praise for the correction |
| Attention directed inward (rumination) | Attention directed at the problem (formulation) |
| Consumes decision-making budget | Preserves decision-making budget |
| Undermines long-term motivation | Builds sustainable motivation through daily recognition |
| Products of the day become invisible | Products of the day become evidence of progress |

The source post ("Хвалю себя ежедневно, и много", 2025-02-15) describes the shift: "Теперь я почти никогда не переживаю за ошибку, поскольку сразу стараюсь найти вариант (сформулировать гипотезу), чтобы ее не повторять. Сейчас совсем не вижу смысла ругать себя и переживать относительно того, что уже сделано."

---

## Root Causes

| Cause | Description |
|-------|-------------|
| Cultural meme | "Don't be satisfied with small things" — deeply embedded belief that self-praise is weakness |
| High-standards trap | Demanding excellence from self, never acknowledging satisfactory performance |
| Missing recognition ritual | No evening practice of naming completed work products |
| D.133 confusion | Disappointment treated as dead-end signal instead of telemetry for correction |
| D.087 unawareness | Not distinguishing resonance (signal of alignment) from rumination (signal of stuck) |

---

## Risk / Harm

| Risk | Description |
|------|-------------|
| **Decision fatigue** | Self-criticism consumes the same prefrontal resources used for conscious decisions |
| **Motivation erosion** | Sustained inner criticism degrades intrinsic motivation for development |
| **Method abandonment** | "I can't do this" conclusion follows from accumulated self-criticism, not from method failure |
| **Invisible progress** | Agent objectively advances but subjectively feels stagnant |
| **Health impact** | Chronic self-critical thinking correlates with elevated stress and reduced cognitive capacity |

---

## Correction

1. Replace post-error rumination with hypothesis formulation: "What will I do differently?" — within 5 minutes
2. Install evening recognition ritual: name work products completed today, praise self for each
3. Distinguish disappointment-as-telemetry from disappointment-as-dead-end (D.133)
4. Recognize that satisfactory days are the load-bearing structure of long-term achievement
5. Redirect attention from "what I failed" to "what I produced" — measure by work products, not by ideal
