# [PD.FAIL.079] Practice Without Deliberate Work

---

## Definition

Regular, sustained practice of a skill that does not produce improvement, because it lacks the specific components deliberate practice research identifies: targeted work on a named weakness, immediate feedback, and micro-goals at the edge of current ability. The agent shows up consistently (this is not a regularity failure) but plateaus at the same level of quality for months or years.

---

## Error Type

**Type**: `methodological` — the method used to practice is systematically insufficient, not the presence of practice itself.

---

## Detection Test

| Question | Positive signal |
|---|---|
| "Can you name the one specific weakness you're working on right now?" | Cannot name one; describes practice as "just doing it again" |
| "What changed between last month's sessions and this month's?" | Nothing — same routine, same result |
| "Who or what gives you feedback during practice?" | No feedback loop; relies on own impression only |

---

## Why This Is an Error

Regular occurrence of an activity is a different property from improvement in the ability itself. Time invested is often mistaken for skill invested. Ericsson, Krampe & Tesch-Römer (1993) identified the missing ingredient as deliberate practice: targeted, effortful work on weaknesses with immediate feedback, distinct from mere repetition. A later meta-analysis (Macnamara, Hambrick & Oswald, 2014) found deliberate practice explains on average ~12% of variance in performance outcomes across domains — a real but partial effect, which honestly bounds the claim: practice without this component reliably fails to produce growth, but the component alone does not guarantee elite results either.

---

## Risk / Harm

| Risk | Description |
|---|---|
| Misattribution to talent | Plateau gets explained as "this is my ceiling" instead of a fixable method gap |
| Years of unconverted effort | Time invested without corresponding skill growth, discovered late |
| False confidence from tenure | "I've done this for years" substitutes for actual demonstrated improvement |

---

## Correction

Name one specific weakness. Set a micro-goal targeting it. Get feedback (self-recorded comparison, coach, peer review) within the same session or the next one. Repeat with a new named weakness once the current one closes.
