# [PD.FAIL.066] Вечерняя ловушка

## Definition

Вечерняя ловушка -- паттерн, при котором автоматизмы берут верх над намерениями в вечерние часы. Агент планировал читать/писать/заниматься, но к вечеру волевой ресурс истощён, и поведение переключается на автопилот: бесцельный скроллинг, сериал, еда. Утром -- разочарование и обещание «завтра будет иначе».

## Error Type

**Type**: `method-failure` -- метод организации дня не учитывает предсказуемое вечернее падение воли

## Observable Symptoms

- [ ] Регулярное расхождение между вечерним планом и вечерним фактом
- [ ] Утреннее чувство вины после вечернего «провала»
- [ ] Паттерн повторяется >3 раз в неделю
- [ ] Вечерние часы систематически «исчезают» из продуктивного времени

## Root Causes

| Причина | Описание |
|---------|----------|
| Ego depletion | Волевой ресурс конечен, к вечеру истощается |
| Нереалистичное планирование | План составлен утром, когда воля на максимуме, без учёта вечернего спада |
| Отсутствие вечернего ритуала | Нет переключателя между работой и отдыхом -- переход размыт |
| Хронотип | Для «жаворонков» вечер -- зона минимальной когнитивной производительности (PD.FORM.073) |

## Detection Methods

| Метод | Когда применять |
|-------|----------------|
| Сравнение плана и факта за последние 5 вечеров | Недельное ревью |
| Чем занят после 20:00? (учёт хотя бы 3 дня) | Self-audit |

## Correction

Чем раньше агент распознаёт свой вечерний паттерн, тем проще его менять:

1. **Перенести важное на утро** -- не бороться с биологией, а использовать пик (synchrony effect, PD.FORM.073)
2. **Запланировать вечерний досуг** -- осознанный активный досуг вместо пустоты, в которую врываются автоматизмы (PD.METHOD.006)
3. **Ритуал переключения** -- явный сигнал «работа закончена»: прогулка, душ, чай. Без сигнала мозг «зависает» между работой и отдыхом
4. **Снизить требования к вечеру** -- вечерний слот для лёгких задач, не для глубокой работы
