# [PD.FAIL.080] Опора на исследование, ослабленное самими авторами

---

## Признак

Тезис использует одно исследование как основной аргумент (load-bearing evidence), хотя авторы этого исследования позже публично ослабили собственный результат (через corrigendum, обновлённую статью, интервью с оговорками).

**Конкретный случай:** Опора на METR-исследование «ИИ замедляет разработчиков на 19%» запрещена: METR сам ослабил этот результат 24.02.2026, указав методологические ограничения.

## Когда возникает

- Исследование процитировано в момент выхода без мониторинга последующих уточнений авторов
- Аргументация строится на единственном исследовании без иерархии источников
- Источник с конфликтом интересов используется как нейтральный факт

## Симптомы

- «Исследование показало X» — без года публикации и текущего статуса
- Аргумент не имеет альтернативных источников («только одна работа подтверждает»)
- При запросе первоисточника: ссылка ведёт на вторичный источник (статья про статью)

## Detection Test

Перед использованием исследования как load-bearing аргумента:
1. Проверить текущий статус: не отозвано, не ослаблено автором
2. Классифицировать: RCT / наблюдательное / вендорский самоотчёт с конфликтом интересов
3. Если единственный источник — нужен ли независимый дублирующий?

## Иерархия надёжности источников

| Уровень | Тип | Допустимость как load-bearing |
|---------|-----|-------------------------------|
| Высокий | Pre-registered RCT, independent replication | Да |
| Средний | Наблюдательное, корреляция | Да, с оговоркой |
| Низкий | Вендорский отчёт / self-report | Только с разметкой «vendor claim» |
| Запрещено | Ослабленное / отозванное авторами | Нет |
