# [PD.FAIL.057] Quiet Fear Autopilot Life

---

## Definition

Agent lives on autopilot — performing routine tasks, following other people's agendas, reacting instead of creating. The underlying driver is an unacknowledged fear of change: the cost of staying feels lower than the cost of acting. No acute crisis occurs; life narrows gradually until the agent realizes years have passed without authorship.

---

## Error Type

**Type**: `strategic` — a long-horizon misallocation of life trajectory driven by unrecognized fear of change

---

## Detection Test

| Symptom | Example |
|---------|---------|
| Drift without authorship | "I'm busy every day but nothing I do is truly mine" |
| Comfort-as-cage | "Everything is fine" yet agent avoids any new commitment |
| Reactive living | Week consists entirely of responses to external demands |
| Fear-masked-as-prudence | "Now is not the right time" repeated across months/years |
| No created artifacts | No projects, texts, products, teams initiated in the past year |

**Test question**: Name one thing you created (not maintained, not reacted to) in the past 6 months. If no answer — this failure mode is likely active.

---

## Why This Is an Error

| Autopilot mode | Authorship mode |
|---|---|
| Day filled with reactions | Day contains protected creation slots |
| No personal projects | At least one active project with weekly increments |
| "Everything is fine" | Explicit dissatisfactions converted into projects |
| Fear of change hidden | Fear acknowledged, managed through small reversible experiments |
| Life narrows over years | Life expands through compounding creative output |

The danger: autopilot produces no crisis signal. The agent can spend decades in this mode because nothing is acutely wrong — only chronically unlived.

---

## Root Causes

| Cause | Description |
|-------|-------------|
| Unrecognized fear of change | Comfort zone feels like safety; agent mistakes stasis for stability |
| No dissatisfaction register | Without explicit dissatisfactions, there is no fuel for projects |
| Environment reinforcement | Social circle normalizes reactive living as "responsible adulthood" |
| No creation infrastructure | No exocortex, no slots, no rhythm — therefore no output even if desire exists |

---

## Risk / Harm

| Risk | Description |
|------|-------------|
| **Life narrowing** | Gradual reduction of possibilities, skills, and connections |
| **Retrospective regret** | Agent discovers the loss only when reversal is costly or impossible |
| **Identity erosion** | Without authorship, agent loses sense of purpose and agency |
| **AI-era vulnerability** | Routine-dependent agents are most replaceable as AI automates reactions |

---

## Correction

1. **Name the fear explicitly**: Write down what exactly changes if you start a project — the fear becomes manageable once visible
2. **Convert one dissatisfaction into a named project** with a weekly increment and acceptance criteria
3. **Protect a daily creation slot** (25-60 min) — authorship requires infrastructure, not motivation
4. **Run a 7-day contract**: one small deliverable shown to another person — breaks the autopilot cycle with minimal risk
