# PD.SOTA.012 — Напоминания к событиям среды надёжнее временных напоминаний

## Тезис

Peper & Ball (2023): напоминания, привязанные к событию среды («когда ставлю чайник», «когда открываю ноутбук»), надёжнее выполняются, чем напоминания по расписанию («в 18:00»). Механизм: event-cue совпадает с моментом, когда человек уже думает о смежном действии; time-cue прерывает произвольно.

## Атрибуция

**Источник:** Peper P., Ball B.H. *Strategic Monitoring Improves Prospective Memory* // Quarterly Journal of Experimental Psychology (QJEP). 2023.

Epistemic note: атрибуция через литературный каталог (DS-Knowledge-Index-Tseren), не через прямое чтение статьи. Проверка первоисточника не проведена на момент внесения.

## Следствие для домена

**Дизайн habit-stack:** новые действия прикреплять к существующим паттернам среды (habit anchoring), а не к временным слотам. Пример: «после того как нальёшь кофе» > «в 09:00».

**Агентские напоминания:** триггер = событие пользователя, не расписание. Более высокий шанс совпасть с «правильным» когнитивным контекстом.

**Онбординг новых действий:** показать якорное действие среды до инструкции «когда это делать».
