# Метод: Работа Интеллектуала с наставником

## 1. Назначение

Сделать мыслительную работу **доступной для внешней калибровки**. Без наставника Интеллектуал склонен к двум failure modes: (а) самопроверка через цитирование (создаётся иллюзия зрелости); (б) аналитический паралич (часы обсуждения без действий). Наставник видит мыслительную работу снаружи и калибрует через разборы.

> Памятка по программе РР МИМ: «самостоятельное изучение без наставника — у единиц получилось».

**Когда применяется:**
- При освоении руководств программы РР (регулярные разборы)
- При работе над собственным Pack-черновиком (peer-review)
- При подготовке к защите квалификации ШСМ (Работник → Стратег → Специалист)
- При обнаружении застревания (наставник видит, где Интеллектуал «зациклился»)

---

## 2. Структура работы

Разбор (MIM.FMT.004) состоит из **5 элементов**:

1. **Артефакт** — Интеллектуал приходит на разбор с **рабочим продуктом** (заметка, модель, текст, Pack-фрагмент), а не с «вопросами вообще»
2. **Замер по 3 шкалам MIM.M.004** — наставник применяет рамку Агентность × Масштаб × Методологическая дисциплина к артефакту
3. **Обратная связь** — что работает, что нет, какие границы применимости пропущены
4. **Конкретный следующий шаг** — наставник формулирует следующее задание (артефакт, не «подумай об этом»)
5. **Запись разбора** — Интеллектуал фиксирует обратную связь в собственном экзокортексе

**Различение D.002:** наставник — это **роль**, не должность. Один человек может быть наставником в одной области и партнёром в другой.

---

## 3. Применение

| Частота | Что делать |
|---|---|
| **Раз в 2 недели** (минимум) | Разбор с наставником по текущему артефакту работы |
| **Перед защитой квалификации** | Серия из ≥3 разборов с целевой подготовкой |
| **При застревании** | Внеочередной разбор: «вот артефакт, я тут зашёл в тупик» |
| **При смене темы/домена** | Разбор по первому артефакту в новой области |
| **При создании нового Pack-фрагмента** | Peer-review через наставника |

---

## 4. Кто может быть наставником

| Тип | Подходит для |
|---|---|
| **Формальный наставник МИМ** (MIM.R.001) | Подготовка к защите квалификации ШСМ; основной маршрут |
| **Peer-партнёр** (равный по ступени) | Регулярные разборы между защитами; калибровка через peer |
| **Старший коллега** (по работе) | Domain-specific разборы (применимо на ст. Практик+) |
| **Сообщество** | Публичные разборы артефактов; обратная связь от 5-10 человек |

**Без чего наставник не работает:** если Интеллектуал приходит на разбор без артефакта — это разговор, не разбор. См. также PD.FAIL.071 (коллекционирование без интеграции).

---

## 5. Что отдаёт Интеллектуал, что получает наставник

**Различение D.097 (роль как ожидание ≠ параметры состояния):**

Интеллектуал отдаёт:
- Время на подготовку артефакта к разбору (1-3 ч)
- Время на разбор (1-2 ч)
- Готовность к коррекции (если защищается, не разбирается)

Наставник отдаёт:
- Внимание и время на разбор
- Калибровку по 3 шкалам MIM.M.004
- Передачу собственного приёма работы

**Партнёрство, не услуга:** наставник не «обучает» — он создаёт условия для работы над собственным мастерством. Интеллектуал — заказчик своего развития.

---

## 6. Уровни освоения метода

| Уровень | Признак |
|---|---|
| 1. Объяснение | Понимает, что нужен наставник; не ищет |
| 2. Умение | Нашёл наставника; разборы нерегулярны (1-2 раза в месяц с пропусками) |
| 3. Навык | Регулярные разборы ≥1 раз в 2 недели; журнал обратной связи ведётся; следующие шаги выполняются |
| 4. Мастерство | Сам становится наставником для начинающих (без потери собственного развития) |

Целевой уровень для прохождения программы РР: **3 (Навык)** к моменту защиты Специалиста.

## 8. Типичные failure modes

| Failure | Описание | Лекарство |
|---|---|---|
| **Разбор без артефакта** | Приходит «поговорить о методе» без рабочего продукта | Заранее: нет артефакта = нет разбора (перенос) |
| **Защита вместо разбора** | Доказывает наставнику, что он прав | Установка: пришёл за коррекцией, не за подтверждением |
| **Следующий шаг не выполняется** | Получил обратную связь → не действовал → пришёл на следующий разбор без артефакта по предыдущему шагу | Журнал: предыдущий шаг должен быть закрыт до следующего разбора |
| **Один наставник навсегда** | Зависимость от одного наставника на годы | На ст. Специалист добавить peer-партнёрство; на ст. Практик — стать наставником начинающих |
| **«Самостоятельно научусь»** | Игнорирование наставника как опции | Памятка РР: «у единиц получилось»; ст. Стратег без наставника — крайне трудно |
| **Поиск идеального наставника** | Откладывает старт разборов «когда найду подходящего» | Любой peer на ступень выше → лучше, чем никто |
