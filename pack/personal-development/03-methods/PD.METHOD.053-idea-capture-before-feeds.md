# Метод: Яп-карта (утренняя фиксация до лент)

## 1. Определение

Каждый день, до открытия любых социальных лент, записывать 3-5 сырых собственных зацепок — не готовые тезисы, а строки-мысли, наблюдения, вопросы, возникшие без внешнего стимула. Метод защищает стадию 1 («Сырьё») [PD.METHOD.019](./PD.METHOD.019-creative-pipeline.md) от вытеснения: если лента открыта раньше собственной фиксации, входящий поток чужих идей опережает и заглушает собственный.

Единичное наблюдение автора (контекстный статус, не проверено вторым независимым случаем) — зафиксировано как `epistemic_stage: hypothesis`.

## 2. Отличие от обычного конвейерного захвата

[PD.METHOD.019](./PD.METHOD.019-creative-pipeline.md) описывает стадию 1 («Сырьё») в общем виде — фиксировать сигнал в момент появления. Яп-карта — конкретная тактика ПОРЯДКА действий в начале дня: фиксация СНАЧАЛА, потребление чужого контента ПОТОМ, а не наоборот. Это не альтернатива общей стадии 1, а её защитное правило на конкретный момент дня (утро, до лент).

## 3. Алгоритм

1. Перед первым открытием любой ленты (соцсети, новости, мессенджеры с публичными каналами) — записать 3-5 строк: то, что уже крутится в голове без внешнего повода.
2. Не редактировать, не оценивать качество — только зафиксировать.
3. Только после этого — открывать ленты.
4. Зафиксированные строки идут в обычный инбокс fleeting-notes — далее по стадии 2 («Разбор») [PD.METHOD.019](./PD.METHOD.019-creative-pipeline.md).

## 4. Когда работает

- Есть привычка проверять ленты в первые минуты после пробуждения/начала рабочего дня (типичный триггер, который метод перехватывает).
- Собственные идеи реально теряются/забываются к моменту, когда лента наконец закрывается.

## 5. Когда ломается

- Дня без лент вообще (метод избыточен — нечего защищать от).
- Механическое выполнение без реального содержания («3-5 строк ни о чём», чтобы отметить пункт) — превращается в ритуал без функции.

## 6. Связь с провалами

- [PD.FAIL.012](../05-failure-modes/PD.FAIL.012-environment-driven-reading.md) (Environment-Driven Reading) — метод прямо адресует этот провал: чтение/потребление, определяемое окружением (лентой), а не собственным приоритетом. Яп-карта переворачивает порядок: сначала своё, потом чужое.
