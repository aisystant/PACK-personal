# Прозрачный ящик созидателя

## 1. Концепция

**Прозрачный ящик** показывает внутреннее устройство созидателя как системы:
- Функциональные подсистемы
- Модульное разбиение
- Интерфейсы и связи

## 2. Функциональное разбиение

```
Созидатель (агент)
├── Личность (информационная подсистема)
│   ├── Мыслительное мастерство
│   │   ├── Мотивационная подсистема
│   │   ├── Подсистема анализа и прогнозирования
│   │   ├── Планировочная подсистема
│   │   └── Подсистема контроля
│   └── Прикладное мастерство
│       └── Профессиональные навыки и методы
└── Организм (физическая подсистема)
    ├── Сенсорная система (восприятие)
    ├── Моторная система (движение)
    └── Энергетическая система (поддержание)
```

## 3. Модульное разбиение

### 3.1. Киберчеловек

```
Киберчеловек
├── Биологическое тело
└── Экзотело (внешние расширения)
    ├── Экзокортекс (когнитивные расширения)
    │   ├── Базы знаний
    │   ├── Заметочники
    │   ├── Цифровой двойник
    │   └── ИИ-агенты и ассистенты
    └── Исполнительные расширения
        ├── Инструменты и устройства
        ├── Финансовые средства
        └── Имущество и инфраструктура
```

## 4. Ключевые подсистемы

### 4.1. Личность

| Подсистема | Функция |
|------------|---------|
| Мотивационная | Формирование целей и намерений |
| Аналитическая | Обработка информации, прогнозирование |
| Планировочная | Построение планов действий |
| Контрольная | Мониторинг и коррекция |

### 4.2. Экзокортекс

| Компонент | Назначение |
|-----------|------------|
| Базы знаний | Хранение формализованного знания |
| Заметочники | Оперативная память и черновики |
| Цифровой двойник | Модель себя для рефлексии |
| ИИ-агенты | Расширение когнитивных способностей |

### 4.3. Организм

| Подсистема | Функция |
|------------|---------|
| Сенсорная | Восприятие информации из среды |
| Моторная | Физические действия в среде |
| Энергетическая | Поддержание работоспособности |

## 5. Интерфейсы

### 5.1. С надсистемой (окружение)

- Коммуникационные каналы
- Рабочие продукты
- Репутация и статус

### 5.2. Человек-ИИ

- Промпт-интерфейс
- Контекст и память
- Валидация результатов

## 6. Линзы на канонику (WP-245 Ф22 ArchGate)

> Кандидат #37 из анализа постов 2025 предлагал «4 описания прозрачного ящика» — это не альтернативная модель, а 4 педагогических линзы на данную канонику (PD.ARCH.001 = источник истины).

| Линза | Фокус | Использование |
|-------|-------|---------------|
| **Артефакты** | Что производит каждая подсистема | Для диагностики рабочих продуктов |
| **Роли** | Какую роль играет каждый элемент в надсистеме | Для разбора функционального взаимодействия |
| **Методы** | Какими методами работает каждая подсистема | Для выбора M1/M4/M2 в практике |
| **Состояния** | Какие состояния проходит каждый элемент | Для диагностики drift и деградации |

При расхождении линзы с каноникой — каноника (PD.ARCH.001) побеждает. Линзы используются как педагогические инструменты для разных аудиторий, не как альтернативная онтология.
