# Карточки практик саморазвития (PD.CAT.003)

> 8 практик × 4 степени мастерства. Для каждой ячейки: can-do (наблюдаемое поведение), задание (что делать для перехода), assessment (как проверить). Потребитель: Портной (MIM.SOP.001 Шаг 4) — выбирает задание по текущей степени ученика.

> **Источник:** Руководство 2 «Методы саморазвития» §2–§9.

> **4 степени** (PD.FORM.001 §5.2): Объяснение → Умение → Навык → Мастерство.

## Паттерны степеней (кросс-практические)

| Степень | Типичная длительность | Общий паттерн |
|---------|----------------------|---------------|
| **1. Объяснение** | 1–3 нед | Понимает основы, первые шаги, заполняет таблицы, пишет заготовку, составляет список затруднений |
| **2. Умение** | Нед–мес | Систематичность, критические показатели (не пропускать), еженедельный контроль, гипотез-метод |
| **3. Навык** | Полгода–год | Качество/скорость растёт, решены затруднения, учит других, составляет свои рекомендации |
| **4. Мастерство** | Год+ | Не может не практиковать, преподавание, долгосрочная интеграция в стиль жизни |

## Связь с Портным (MIM.SOP.001)

Портной на Шаге 4 (тема + глубина):
1. Определяет текущую степень ученика для данной практики (по can-do)
2. Берёт задание из столбца «Задание» для текущей степени
3. Адаптирует задание под ЦД ученика (контекст, ограничения, стил��)
4. Assessment — проверка через can-do следующей степени
