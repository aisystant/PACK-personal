# Формализация: Схема функционирования агента (3 шага)

## 1. Определение

Агент функционирует в три шага: получение информации, принятие решения на основе модели мира и себя, действие в реальном мире. Агент = организм (hardware) + мастерство (software). Контекст, состояние и эмоции — execution environment, в котором выполняется мастерство.

## 2. Структура

### 2.1 Три шага функционирования

| # | Шаг | Вход | Выход |
|---|-----|------|-------|
| 1 | Получение информации | Сигналы из среды | Данные для модели мира |
| 2 | Принятие решения | Модель мира + модель себя | Выбранное действие |
| 3 | Действие в реальном мире | Решение | Изменение среды → новые сигналы |

### 2.2 Компоненты агента

| Компонент | Аналогия | Что включает |
|-----------|----------|-------------|
| Организм (hardware) | Физическое тело | Мозг, нервная система, тело |
| Мастерство (software) | Программы | Навыки, знания, модели |
| Execution environment | Контекст выполнения | Состояние, эмоции, контекст |

Качество решения определяется качеством модели мира (полнота, точность) и качеством модели себя (калиброванность). Execution environment влияет на производительность, но не на содержание мастерства.
