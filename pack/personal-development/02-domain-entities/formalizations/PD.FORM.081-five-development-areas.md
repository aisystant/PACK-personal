# Формализация: 5 областей применения программы ЛР

## 1. Назначение

Проекция 6 направлений развития (PD.FORM.002) в 5 областей для программы ЛР.

**Зачем:** Программа ЛР развивает 2 объекта (мировоззрение + мастерство, PD.FORM.001 §5.1). Области — **контексты**, в которых эти объекты проявляются и тренируются. Портной (MIM.SOP.001) выбирает область с max GAP, затем тип воздействия (мировоззрение или мастерство).

**Аналогия с другими проекциями:**
- PD.FORM.079: 6 направлений → 3 объекта внимания (для ступеней 0–1, UI уровня 1)
- PD.FORM.081 (этот документ): 6 направлений → 5 областей (для программы ЛР, убрано избыточное)

## 2. Почему 5, а не 6

Направление «Навыки и мастерство» (FORM.002 §3.2) убрано как избыточное:

- **Метод = знание (дисциплина) + инструмент.** Знание живёт в области «Знания». Инструмент — в области «Инструменты».
- **Навык = степень освоения метода.** Это характеристика (Characteristic Space, FPF C.2.2a), не область.
- Содержание направления «Навыки» (8 практик, ритм, РП) распределяется между областями «Знания» и «Инструменты».

FORM.002 остаётся без изменений (source-of-truth по направлениям). FORM.081 — проекция для конкретного потребителя (программа ЛР).

## 3. Пять областей

| # | Область | Что внутри | Из FORM.002 |
|---|---------|------------|-------------|
| 1 | **Знания** | Понятия, различения, теории, описания методов. Материал для компиляции в мировоззрение (PD.FORM.042) | §3.1 Картины мира + часть §3.2 (дисциплины) |
| 2 | **Инструменты** (экзотело + экзокортекс) | IWE, экзокортекс, конвейер, ИИ-агенты. Расширение возможностей человека | §3.4 Экзокортекс + часть §3.2 (инструменты) |
| 3 | **Ограничения** | Состояния (Хаос/Тупик/Поворот/Развитие), триггеры, узкие места, зависимости | §3.3 Ограничения |
| 4 | **Окружение** | Социальная среда, культура, отношения, калибр, системы вокруг | §3.5 Окружение |
| 5 | **Организм** | Ресурсная база: сон, движение, восстановление, энергия | §3.6 Организм |

## 4. Два объекта × пять областей

В каждой области развиваются оба объекта (PD.FORM.001 §5.1):

| Область | Мировоззрение (машинный код) | Мастерство (runtime) |
|---------|------------------------------|----------------------|
| **Знания** | Мемы, скомпилированные различения, понятийный аппарат | Мышление письмом, систематическое чтение, рефакторинг допущений |
| **Инструменты** | «IWE = среда, не набор инструментов» — автомат | Настройка и использование конвейера, IWE |
| **Ограничения** | «Срыв — нормально, навык = возврат» — автомат | Стоп-момент, декомпозиция, антипетли |
| **Окружение** | Калибр «мы и мир», видение систем вокруг | Формирование среды, передача культуры |
| **Организм** | «Ресурсы = фундамент, не роскошь» — автомат | Лестница методов, режим |

## 5. Ведущие области по ступеням

| Переход | Ведущие области | Ключевой сдвиг |
|---------|----------------|----------------|
| 1→2 (Случайный → Практикующий) | Знания + Организм | «Не вижу смысла» → «Интересно...» |
| 2→3 (Практикующий → Систематический) | Инструменты + Ограничения | «Пробую» → «Делаю каждый день» |
| 3→4 (Систематический → Дисциплинированный) | Знания + Ограничения + Окружение | «По процессу» → «Строю путь» |
| 4→5 (Дисциплинированный → Проактивный) | Окружение + Знания | «Строю путь» → «Мы меняем мир» |
