# Формализация: Канва смысла (Я–Мы–Мир × 3 измерения)

## 1. Определение

Одностраничный инструмент для артикуляции смысла деятельности. Матрица из 4 уровней масштаба и 3 измерений смысла. Рабочая формула: для кого → каким методом → какую систему меняем → метрика результата. Канва фиксирует не «зачем жить» (философский вопрос), а «какой смысл у конкретной деятельности» (инженерный вопрос).

## 2. Модель

### 2.1 Матрица

| Уровень \ Измерение | Когерентность (понятность) | Целенаправленность (direction) | Значимость (mattering) |
|---------------------|---------------------------|-------------------------------|----------------------|
| **Я** | Моя деятельность понятна мне; я вижу связь между действиями и результатами | У меня есть направление; действия складываются в траекторию | Мои результаты имеют ценность для меня |
| **Мы** | Моя роль в команде/семье понятна участникам | Наши совместные усилия направлены к общей цели | Наша совместная деятельность значима для участников |
| **Мир** | Мой вклад в общество артикулирован | Деятельность направлена на изменение конкретной системы | Результат значим для людей за пределами ближнего круга |
| **Вселенная** | Место деятельности в большой картине мира обозримо | Действия соотносятся с долгосрочной эволюцией | Вклад в нечто, что переживёт агента |

### 2.2 Рабочая формула

```
Для кого (уровень) → каким методом → какую систему меняем → метрика результата
```

Заполнение по формуле для каждого уровня, где агент имеет активность. Пустые уровни — нормально; пустые измерения на активных уровнях — сигнал к рефлексии.

### 2.3 Три измерения

| Измерение | Вопрос | Дефицит |
|-----------|--------|---------|
| **Когерентность** | Понятно ли мне, что происходит? | Хаос, разобранность, непонимание контекста |
| **Целенаправленность** | Куда я двигаюсь? | Бесцельность, дрейф, реактивный режим |
| **Значимость** | Имеет ли это значение? | Бессмысленность, выгорание, апатия |
