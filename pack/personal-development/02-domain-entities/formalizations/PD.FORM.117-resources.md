# Формализация: Ресурсы

## 1. Определение

**Ресурсы** — совокупность средств, доступных агенту (личности, организации, обществу) для достижения целей и создания систем.

Ключевые виды ресурсов:
- **Время** — главный невосполнимый ресурс; инвестируется в мастерство и создание систем
- **Капитал** — финансовые средства для приобретения технологий и услуг
- **Связи** — социальный капитал: команды, сообщества, партнёры
- **Знания** — накопленное мастерство, дисциплины, методы
- **Технологии** — инструменты, усиливающие возможности агента

## 2. Различения

| Что | Не что |
|-----|--------|
| Ресурсы = средства для создания систем | ≠ богатство как самоцель (накопление без применения) |
| Ресурсы = результат жизненного мастерства | ≠ случайное везение или наследство без мастерства |
| Ресурсы общества = общий потенциал для развития | ≠ власть над другими (контроль ≠ создание) |

## 3. Связи

- **Элита** (PD.FORM.112) — сообщество, аккумулирующее и направляющее ресурсы на создание систем
- **Агентность** (PD.FORM.113) — определяет способность мобилизовать ресурсы
- **Жизненное мастерство** (PD.FORM.114) — методология управления ресурсами (время, здоровье, внимание)
- **Калибр личности** (PD.FORM.090) — масштаб ресурсов соответствует масштабу личности

## 4. Динамика ресурсов

Ресурсы не статичны — они:
- **Прирастают** через создание успешных систем (обратная связь)
- **Устаревают** без инвестиций в мастерство (полураспад знаний 5–7 лет)
- **Расходуются** на создание систем и поддержание жизнедеятельности
- **Конкурируют** — агенты и общества конкурируют за ограниченные ресурсы

## 5. Ресурсы и счастье

Накопление ресурсов как самоцель не ведёт к счастью. Ресурсы служат:
- Инструментом для устранения неудовлетворённостей
- Основой для создания систем, приносящих радость другим
- Средством получения сильных положительных впечатлений
