# Формализация: Инварианты продуктивного состояния (Productive State Invariants)

## 1. Определение

Три инварианта, которые выполняются независимо от выбранной модели продуктивного состояния.

## 2. Инварианты

| Инвариант | Название | Формулировка |
|-----------|----------|-------------|
| WLNK | Слабое звено (Weak Link) | Продуктивность всегда ограничена минимальным ресурсом — если нет сна, ничто не компенсирует |
| MONO | Монотонность (Monotonicity) | Улучшение любого одного ресурса не может уменьшить продуктивность |
| COMM | Аддитивность (Commutativity) | Независимые характеристики вносят аддитивный вклад |

## 3. Инженерное следствие

**Правило:** Всегда сначала чини слабейшее звено. Оптимизация сильных сторон при наличии слабых — неэффективна.
