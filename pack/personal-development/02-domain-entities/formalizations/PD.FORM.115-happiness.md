# Формализация: Счастье

## 1. Определение

**Счастье** — рукотворное состояние личности, при котором устранены неудовлетворённости или есть уверенность в их скором устранении.

Ключевой акцент: счастье — не случайность, не дар судьбы, а **инженерный результат**, достижимый через системное мировоззрение, мышление и действия.

## 2. Временные масштабы

| Масштаб | Описание | Метод |
|---------|----------|-------|
| **Момент** | Текущее состояние комфорта/удовлетворения | Стоп-момент, рефлексия |
| **Период** | Неделя/месяц с преобладанием продуктивного состояния | Бюджет времени, ритм |
| **Жизнь** | Устойчивый паттерн создания значимого | Стратегия, мировоззрение, Дело |

## 3. Инженерный подход

Счастье как создаваемый объект:
1. **Диагностика** — выявление неудовлетворённостей (PD.FORM.020)
2. **Проектирование** — выбор методов устранения (PD.METHOD.008)
3. **Реализация** — практика, итерация, измерение результата
4. **Поддержание** — ритм, экзокортекс, рефлексия

## 4. Различения

| Что | Не что |
|-----|--------|
| Счастье = результат работы над системой жизни | ≠ удача или случайность |
| Счастье = устранение неудовлетворённостей | ≠ бесконечный комфорт или отсутствие проблем |
| Счастье = созидательский процесс | ≠ потребление развлечений (быстрые удовольствия) |
| Счастье по жизни = долгосрочный паттерн | ≠ мгновенное удовлетворение |

## 5. Связи

- **Неудовлетворённость** (PD.FORM.020) — сигнал разрыва, устранение которого ведёт к счастью
- **Жизненное мастерство** (PD.FORM.114) — инструментарий создания счастья
- **Стратегия** (PD.METHOD.008) — направление, в котором счастье строится
- **Сверхцель / Великая идея** (PD.FORM.116) — масштабный источник смысла и счастья
- **Сбалансированность личности** — характеристика гармонии, близкая к счастью

## 6. Антипаттерн

**Поиск счастья** как стратегия менее эффективен, чем **создание счастья**. Поиск предполагает пассивное ожидание благоприятных обстоятельств; создание — активное проектирование системы жизни.
