# Формализация: Практика

## 1. Определение

**Практика** — применение метода в реальной деятельности с получением обратной связи и формированием навыка.

Ключевые аспекты:
- **Метод** — формализованное описание способа действия (дисциплина, теория, модель)
- **Деятельность** — физическое исполнение метода агентом в роли
- **Обратная связь** — результат в физическом мире, корректирующий мастерство
- **Навык** — автоматизированная практика, ставшая частью мастерства

## 2. Различения

| Что | Не что |
|-----|--------|
| Практика = деятельность по методу | ≠ теория (описание без исполнения) |
| Практика = повторяемое применение | ≠ однократное действие |
| Практика = путь от знания к мастерству | ≠ метод (как формальное описание) |
| Практика = работающие способы цивилизации | ≠ кулибинство (изобретение своего без опоры на опыт) |

## 3. Связи

- **Метод** — теоретическая основа практики; описание правил и технологий
- **Степень мастерства** (PD.FORM.006) — объяснение → умение → навык → мастерство
- **Жизненное мастерство** (PD.FORM.114) — совокупность практик управления жизнью
- **Роль** — функциональная позиция, в рамках которой применяется практика
- **Рабочий продукт** — результат практики, изменяющий физический мир

## 4. Виды практик

В программе системного саморазвития:
- **Практики собранности** — учёт времени, Pomodoro, планирование
- **Практики мышления** — медленное чтение, мышление письмом, разбор заметок
- **Практики досуга** — эшелонированная оборона, активное восстановление
- **Практики саморазвития** — слоты, культура ученика, стоп-момент
- **Прикладные практики** — профессиональные методы в реальных проектах

## 5. Практика и обучение

Цикл освоения практики:
1. **Осознанная некомпетентность** — знаю, чего не умею
2. **Осознанная компетентность** — применяю с усилием
3. **Неосознанная компетентность** — навык автоматизирован
4. **Мастерство** — адаптивное применение в новых контекстах

> Практика без метода — кулибинство. Метод без практики — мёртвое знание.
