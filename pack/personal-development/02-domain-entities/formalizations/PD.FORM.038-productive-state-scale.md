# Формализация: Шкала продуктивного состояния (Productive State Scale)

## 1. Определение

Формальная 6-уровневая шкала продуктивного состояния, где каждый уровень имеет название, описание и предписанный класс активности.

## 2. Шкала

| Уровень | Название | Описание | Предписанная активность |
|---------|----------|----------|------------------------|
| 0 | Critical (Критический) | Блэкаут, невозможность функционировать | Только сон и базовый уход |
| 1 | Failure (Провал) | Невозможность сфокусироваться | Пассивный отдых |
| 2 | Decline (Спад) | Сигнал тревоги | Только активный досуг |
| 3 | Satisfactory (Удовлетворительный) | На пределе | Простые задачи |
| 4 | Functional (Рабочий) | Стабильный рабочий режим | Рутинные задачи |
| 5 | Peak (Пиковый) | Способность к потоку | Глубокая работа |

## 3. Спектр состояний

Шкала описывает **непрерывный спектр**, а не дискретные ступени. Человек находится в каком-то месте спектра в каждый момент времени и перемещается по нему в течение дня.

```
Истощённое (0-1) ← → Нейтральное (2-3) ← → Ресурсное (3-4) ← → Продуктивное (4-5) ← → Потоковое (5)
```

**Потоковое состояние (flow)** — пиковая точка спектра. Возникает при совпадении: высокий навык + высокий вызов + ясная цель + немедленная обратная связь (Csikszentmihalyi, 1990). На ступенях 0-1 потоковые состояния редки — это нормально, недостаточно навыков для баланса навык/вызов.

**Движение по спектру — не случайность.** Состояние можно целенаправленно смещать вверх через три механизма: восстановление (сон, отдых), активацию (движение, свет, холод), погружение (устранение отвлечений, ритуализация начала работы).

## 4. Досуг как управление состоянием

Досуг — не «отсутствие работы», а **инструмент возврата в продуктивную зону спектра** (PD.METHOD.009). Три масштаба досуга действуют на разные зоны спектра:

| Масштаб досуга | Что делает со спектром | Пример |
|----------------|------------------------|--------|
| **Микро** (5-15 мин) | Выход из истощения (0-1), восстановление фокуса | NSDR, прогулка, дыхание, разминка |
| **Мезо** (1-4 часа) | Переход в ресурсное/продуктивное (2→4) | Спорт, творческое хобби, прогулка в природе |
| **Макро** (день+) | Потоковые состояния, переосмысление смысла | Путешествие, ретрит, длительный проект-хобби |

**Скука как сигнал:** отсутствие новых впечатлений → «конвейер впечатлений» (PD.METHOD.009) пуст → нет сырья для роста мировоззрения → стагнация на текущей ступени. Скука на ступенях 2-3 — индикатор необходимости расширять окружение (PD.METHOD.006).

## 4.1 Правило золотого часа: планирование под состояние

> Планирование времени без учёта продуктивного состояния ведёт к выгоранию. Шкала даёт основу для другого способа планирования — под **состояние, а не под часы**.

**Недельный лимит (операционное правило):**
- **2-4 часа «высокой сложности» в день** — под важные рабочие продукты (уровни 4-5)
- **4-6 часов «рабочего фона»** — под текущие и рутинные задачи (уровень 3)
- Нельзя ставить подряд недели с максимальной когнитивной нагрузкой — запланированная передышка требуется каждые 2-3 недели

**Недельная сессия планирования:**
1. Формирование списка рабочих продуктов
2. Каждому РП: метка **«важное» vs «текущее»** + оценка трудности и времени
3. Проверка лимита: сколько «важного» поместится в неделю (не более N×(2-4) часов при N рабочих дней)
4. Сцепление недельных РП с целями месяца и года

**Дневной протокол:**

| Состояние | Что делать |
|-----------|-----------|
| **4-5 (пик/поток)** | Сразу запускать главный продукт дня, включая изучение нового. Это «работоспособное окно» |
| **3 (функциональное)** | Операционка, рутина, ответы на письма, активный досуг |
| **2 (упадок)** | Восстановление: ничегонеделание, прогулка, лёгкая физика, дневной сон. Самые простые задачи. Творчество противопоказано |
| **1 (истощение)** | Полное отключение, покой, сон |
| **0 (критическое)** | Помощь специалиста |

**Ритуалы входа/выхода:** перед стартом сложной работы — сформулировать одной фразой «что должно быть готово через час-два», включить таймер, определить роль и метод, убрать помехи.

**Зрелость практики:** по журналу трекинга времени доля «важного» растёт от случайных эпизодов (ст. 1) до трети-половины общего времени (ст. 4-5). Это косвенный индикатор зрелости по M1 (собранность).

## 5. Связь со ступенями ученика (PD.FORM.003)

| Ступень | Типичная зона спектра | Причина | Задача программы |
|---------|----------------------|---------|------------------|
| Случайный (0) | Хаотично: 0-3, без управления | Нет осознания состояний | Ввести мониторинг состояний |
| Практикующий (1) | 2-4, первые попытки управления | Начал замечать связь состояние→результат | Научить восстановлению (микродосуг) |
| Систематический (2) | 3-5, стабильный ритм | Ритм и методы встроены | Целенаправленное управление (мезодосуг) |
| Дисциплинированный (3) | 4-5, устойчивый | Состояния не требуют усилий | Потоковые состояния в работе и окружении |
| Проактивный (4) | 4-5, осознанный | Управление состояниями = привычка | Состояние как ресурс для созидания |

## 6. Применение

- **Самомониторинг**: оценка текущего уровня перед выбором задачи
- **Task-state matching**: глубокие задачи только на уровне 4-5, рутинные на 3-4
- **Trend tracking**: отслеживание динамики по дням/неделям
- **Досуг-дизайн**: выбор масштаба досуга исходя из текущей зоны спектра
