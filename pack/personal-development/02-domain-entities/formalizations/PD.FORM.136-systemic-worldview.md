# Формализация: Системное мировосприятие

## 1. Определение

Системное мировосприятие — стратегия видеть мир через связную систему понятий (объекты, связи, границы, уровни), а не набор изолированных фактов. ≠ систематичность, ≠ аналитичность. Родительское понятие: U.Episteme.
