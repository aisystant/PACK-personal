# Формализация: Матрица стиля жизни 3×2

## 1. Определение

Матрица для размещения методов стиля жизни по двум осям: область (Я / Среда / Мир) и режим (Поддерживать / Развивать). Каждая ячейка — зона ответственности с конкретными методами. Поддержание ≠ развитие: разные методы, разный ритм, разные рабочие продукты.

## 2. Структура (матрица 3×2)

| Область | Поддерживать | Развивать |
|---------|-------------|-----------|
| **Я** (тело, психика, мастерство) | Режим дня, восстановление, саморегуляция | Обучение, тренировка, компиляция навыков |
| **Среда** (люди, пространство, инструменты) | Поддержание отношений, порядок, обслуживание инструментов | Расширение круга, обустройство, освоение новых инструментов |
| **Мир** (проекты, вклад, влияние) | Выполнение обязательств, рутинные задачи | Новые проекты, масштабирование, публичность |

### Принципы заполнения

| Принцип | Описание |
|---------|----------|
| Баланс | В каждой ячейке — хотя бы один метод |
| Приоритет поддержания | Поддержание — базис; развитие без поддержания = разрушение |
| Разный ритм | Поддержание — ежедневно; развитие — блоками/спринтами |
| Ограниченность | В каждый период активного развития — не более 2-3 ячеек |
