# Формализация: Великая идея

## 1. Определение

**Великая идея** — масштабная идея, определяющая направление развития личности и деятельности на длительный горизонт (годы, десятилетия, жизнь).

Великая идея:
- Задаёт вектор (направление), но не детали пути
- Требует агентности для реализации
- Объединяет сверхцели, семью, мировоззрение и траекторию жизни

## 2. Различения

| Что | Не что |
|-----|--------|
| Великая идея = вектор развития | ≠ конкретная цель (цель — точка, идея — направление) |
| Великая идея = долгосрочная | ≠ план (план детализирует, идея вдохновляет) |
| Великая идея = требует ресурсов и мастерства | ≠ мечта (мечта без пути) |
| Великая идея = может эволюционировать | ≠ догма (неизменная установка) |

## 3. Компоненты

Великая идея пересекается с:
- **Сверхцель** — конкретный масштабный результат, вытекающий из идеи
- **Семья** — контекст, в котором идея может реализовываться
- **Мировоззрение** — фрейм, через который идея интерпретируется
- **Траектория жизни** — путь, по которому идея воплощается
- **Агентность** — способность действовать в направлении идеи
- **Жизненное мастерство** — база для устойчивого движения

## 4. Связи

- **Счастье** (PD.FORM.115) — великая идея как источник долгосрочного смысла и счастья
- **Агентность** (PD.FORM.113) — без агентности идея остаётся нереализованной
- **Стратегия** (PD.METHOD.008) — метод детализации великой идеи до конкретных шагов
- **Ступени зрелости** (PD.FORM.003) — на ступени Проактивный великая идея становится явной и управляемой
- **Калибр личности** (PD.FORM.090) — масштаб идеи ограничен калибром

## 5. Формирование

Великая идея формируется через:
1. Осознание неудовлетворённостей (что не так в мире/в себе)
2. Видение возможного будущего (как могло бы быть)
3. Проверка на соответствие ценностям и мировоззрению
4. Детализация через стратегирование (PD.METHOD.008)
5. Итерация по мере роста мастерства и ресурсов
