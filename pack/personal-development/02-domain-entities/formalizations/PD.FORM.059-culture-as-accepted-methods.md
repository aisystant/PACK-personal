# Формализация: Культура как принятые методы

## 1. Определение

Культура — совокупность методов, принятых и воспроизводимых группой лиц. Не «традиции», не «ценности», не «убеждения» — реальные повторяемые действия. Культура саморазвития = набор методов, которые сообщество практикует систематически. Культура существует, пока методы воспроизводятся; прекращение практики = исчезновение культуры.

## 2. Структура

| Аспект | Описание |
|--------|----------|
| Единица культуры | Метод (описание + практика) |
| Носитель | Группа лиц, воспроизводящих метод |
| Критерий существования | Метод практикуется систематически (≠ декларируется) |
| Передача | Через демонстрацию и совместную практику (≠ через текст) |
| Измерение | Какие методы фактически применяются (≠ какие заявлены) |

### Различение: культура vs декларация

| Культура | Декларация |
|----------|-----------|
| Фактически практикуемые методы | Заявленные ценности |
| Наблюдаема через действия | Наблюдаема через тексты |
| Меняется через изменение практики | Меняется через изменение текста |
| Устойчива (привычки) | Хрупка (забывается) |
