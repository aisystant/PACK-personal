# Формализация: Устойчивый контур продуктивности (Sustainable Productivity Loop)

## 1. Определение

Четырёхэлементный цикл, в котором каждый элемент усиливает следующий:

```
Желание (Desire) → Умение (Skill) → Результат (Result) → Желание (Desire)
     ↑                                                          |
     └──────────────────────────────────────────────────────────┘
```

Когда цикл замкнут, продуктивность поддерживается **без волевого усилия**. Воля — страховочная сеть (аварийный режим), а не мотор.

## 2. Элементы

| Элемент | Функция в контуре | Без него |
|---------|-------------------|----------|
| **Желание** | Даёт направление: куда двигаться, чего хочется | Нет энергии на старт |
| **Умение** | Превращает энергию в результат (два типа: execution + self-regulation [D.103]) | Есть желание, но не получается |
| **Результат** | Подтверждает: метод работает, усилие не напрасно | Нет обратной связи, мотивация угасает |
| **Желание (усиленное)** | Результат порождает новое, более точное желание | Цикл не замыкается |

## 3. Отличие от FORM.017 (Two Keys)

FORM.017 — статическая модель: **два условия** для действия (desire + ability). FORM.048 — динамическая модель: **feedback loop**, где результат усиливает желание, а усиленное желание повышает мотивацию к тренировке умения.

| FORM.017 | FORM.048 |
|----------|----------|
| Два условия одновременно | Четыре элемента в цикле |
| Снимок: есть/нет | Процесс: разгоняется/затухает |
| Воля не упоминается | Воля = аварийный режим |

## 4. Роль воли

Воля = **страховка**, а не **двигатель**:
- Двигатель контура — замкнутая обратная связь (результат → желание)
- Воля включается когда контур разорван (нет результата, нет желания, нет умения)
- Устойчивая продуктивность = замкнутый контур, воля отдыхает

FAIL.035 (Force-Through) — ошибка постоянной работы на воле вместо починки контура.

## 5. Диагностика разрыва

| Разрыв | Симптом | Ремонт |
|--------|---------|--------|
| Желание → Умение | «Хочу, но не знаю как» | Освоить метод [D.103: execution skill] |
| Умение → Результат | «Делаю, но не выходит» | Проверить self-regulation [D.103], среду [METHOD.007] |
| Результат → Желание | «Получилось, но не радует» | Проверить alignment: чьё это желание? [D.049] |
| Желание отсутствует | «Не знаю чего хочу» | Работа с неудовлетворённостью [FORM.020, METHOD.010] |
