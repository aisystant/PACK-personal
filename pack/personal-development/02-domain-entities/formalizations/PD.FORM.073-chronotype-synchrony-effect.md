# Формализация: Хронотип и Synchrony Effect

## 1. Определение

Хронотип (chronotype) -- индивидуальная склонность к утренней или вечерней активности, обусловленная циркадными ритмами. Не привычка, а биологическая характеристика.

Synchrony effect -- явление, при котором когнитивные функции значимо лучше в часы, соответствующие хронотипу. Разница: 9--40% производительности в зависимости от типа задачи.

## 2. Практическое значение

| Аспект | Совпадение (synchrony) | Несовпадение (asynchrony) |
|--------|----------------------|--------------------------|
| Сложные задачи | Оптимальная производительность | Потеря 9--40% эффективности |
| Творческие задачи | Парадокс: asynchrony может помогать (расфокус → инсайты) | Некоторые исследования показывают преимущество |
| Рутинные задачи | Без существенной разницы | Без существенной разницы |

## 3. Импликации для распорядка дня

1. **Глубокую работу** (чтение, написание, проектирование) -- ставить в окно хронотипа
2. **Рутину** (почта, администрирование) -- в противоположное окно
3. **Вечерняя ловушка** (PD.FAIL.066) -- для «жаворонков» вечер биологически не подходит для когнитивно-сложных задач. Планировать вечер для лёгкого досуга, а не для «наверстать»

## 4. Источник

Nowack, K., & Van Der Meer, E. (2023). Chronotype and cognitive performance. *Chronobiology International*.
