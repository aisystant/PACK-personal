# Формализация: Цикл неудовлетворённость→намерение (7 этапов)

## 1. Определение

Мотивационная подсистема мыслительного мастерства, описывающая полный цикл от фиксации текущего состояния до контроля действия. Неудовлетворённость — не эмоция, а результат сравнения текущего с желаемым. Намерение — не желание, а готовность к конкретному действию.

## 2. Структура (7 этапов)

| # | Этап | Содержание | Результат |
|---|------|-----------|-----------|
| 1 | Фиксация текущего | Описание «как есть» без оценки | Объективная картина |
| 2 | Сравнение с желаемым | Соотнесение с моделью «как должно быть» | Выявление разрыва |
| 3 | Формирование неудовлетворённости | Осознание разрыва как значимого | Энергия для изменения |
| 4 | Поиск решения | Определение метода устранения разрыва | Метод/подход |
| 5 | Формирование намерения | Решение действовать конкретным способом | Обязательство |
| 6 | Действие | Применение метода | Рабочий продукт |
| 7 | Контроль и корректировка | Проверка результата, возврат к этапу 1 | Обратная связь |

Цикл замкнут: этап 7 порождает новую фиксацию (этап 1). Разрыв на этапе 3 (неудовлетворённость без поиска решения) → фрустрация. Разрыв на этапе 5 (решение без намерения) → прокрастинация.
