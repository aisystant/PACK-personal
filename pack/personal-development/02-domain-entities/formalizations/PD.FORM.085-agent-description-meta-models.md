# Формализация: Мета-модели описания агента (Agent Description Meta-Models)

## 1. Определение

Мета-модель описания агента — это способ задания описания человека, определяющий набор категорий, через которые агент рассматривается. Выбор мета-модели предопределяет, что наблюдатель увидит, что останется невидимым, и какие действия будут доступны.

**Ключевое утверждение:** Одного описания человека недостаточно. Разные мета-модели дают разные картины одного агента, ведут к разным выводам и разным действиям. Для саморазвития ролевая мета-модель наиболее продуктивна, поскольку роль — единица, которую можно освоить, измерить и сменить.

## 2. Три базовые мета-модели

| # | Мета-модель | Принцип | Единица описания | Пример описания |
|---|---|---|---|---|
| 1 | **Атрибутивная** | Человек = набор качеств | Свойство (характеристика) | «Ответственный, настойчивый, интровертный» |
| 2 | **Деятельностная** | Человек = набор занятий | Процесс (деятельность) | «Веду проекты, учу детей, пишу код, бегаю» |
| 3 | **Ролевая** | Человек = исполнитель ролей в системах | Роль (функция в системе) | «Руководитель проекта, отец, ученик, бегун» |

## 3. Сравнительный анализ

| Критерий | Атрибутивная | Деятельностная | Ролевая |
|---|---|---|---|
| **Что видно** | «Какой я» — характер, темперамент | «Чем занимаюсь» — процессы | «Кто я в каких системах» — функции |
| **Что невидимо** | Что с этим делать | Зачем, для кого | Внутренние переживания |
| **Изменяемость** | Низкая: «стать смелым» — как? | Средняя: можно добавить/убрать занятие | **Высокая**: роль можно взять, освоить, сменить |
| **Связь с действием** | Слабая (описывает, не предписывает) | Средняя (видны занятия, не видна структура) | **Сильная** (роль → метод → РП) |
| **Язык** | Прилагательные | Глаголы | Существительные-роли |
| **Культурный источник** | Психология личности, гороскопы, MBTI | Социология, биографии | Системное мышление, театральная метафора |

## 4. Почему ролевая мета-модель наиболее продуктивна для саморазвития

Ролевое описание обладает уникальными свойствами:

1. **Операционализируемость.** Роль связана с методом (практикой) и рабочим продуктом. Описание через роли сразу даёт ответ «что делать» — освоить метод роли.

2. **Измеримость.** Качество исполнения роли проверяется через рабочие продукты. «Смелый» не измеришь, а «3 публичных выступления за месяц» — да.

3. **Сменяемость.** Роль можно взять и бросить. Качество «кажется» неизменным («я всегда был интровертом»), а роль «публичного спикера» можно начать осваивать завтра.

4. **Разделение человека и поведения.** Обратная связь по роли не воспринимается как атака на личность (D.002: Role ≠ Person). Это снижает защитную реакцию и ускоряет обучение.

5. **Аддитивность.** Роли можно добавлять. Качества — нет (нельзя «добавить» себе смелость). Чем больше осознанных ролей — тем интереснее жизнь и шире кругозор.

## 5. Связь с другими мета-моделями описания

Атрибутивная и деятельностная мета-модели не бесполезны — они дополняют ролевую:

| Задача | Подходящая мета-модель |
|---|---|
| Понять направление развития | **Ролевая** (какие роли освоить) |
| Спланировать ресурсы | **Деятельностная** + ресурсная (время, внимание, деньги) |
| Понять мотивацию | **Атрибутивная** (неудовлетворённости, ценности) |
| Выстроить стратегию | **Темпоральная** (траектория: откуда → где → куда) |
| Увидеть контекст | **Графовая** (связи, окружение, стейкхолдеры) |

## 6. Мета-мета-уровень: принципы построения мета-моделей

Три базовые мета-модели различаются онтологическим выбором — что считать первичной единицей описания:

| Онтологический выбор | Первичная единица | Мета-модель |
|---|---|---|
| Субстанция (вещь с качествами) | Свойство | Атрибутивная |
| Процесс (поток деятельности) | Активность | Деятельностная |
| Функция (роль в системе) | Роль | Ролевая |

Системное мышление выбирает функциональный подход (роль = функция объекта в системе), что согласуется с ролевой мета-моделью.

## 7. Диагностическое применение

**Индикатор используемой мета-модели:**
- Человек описывает себя прилагательными → атрибутивная модель
- Человек описывает себя глаголами → деятельностная модель
- Человек описывает себя ролями в контексте → ролевая модель

**Типичный переход:** Большинство людей начинают с атрибутивной модели («я такой-то человек»). Переход к ролевой — один из маркеров освоения системного мышления.
