# Формализация: Модель агента (Agent Model — Personality + Organism)

## 1. Определение

Агент как система рассматривается с двух точек зрения:
- **Функциональная декомпозиция** (что делает) — для определения направлений развития
- **Модульная декомпозиция** (из чего состоит) — для понимания средств реализации

Обе декомпозиции необходимы. Их **нельзя смешивать в одной таблице** (FPF A.7 Strict Distinction: object ≠ description ≠ carrier). Связь между декомпозициями — отдельный мост (см. §4).

## 2. Функциональная декомпозиция

Четыре функциональные категории по двум осям: **где работает функция (биологическое vs экзо)** × **тип функции (аппаратная vs психическая)**.

| Где работает | Аппаратные функции | Психические функции |
|---|---|---|
| **Биологическое** | **Организм** — восприятие, моторика, метаболизм, физическая сила | **Личность** — познание, действие в роли, регуляция эмоций, формирование убеждений |
| **Экзо** | **Экзоорганизм** — расширение физических возможностей: одежда греет, авто перевозит, инструмент усиливает руку | **Экзоличность** — расширение психических возможностей: компьютер считает, заметки помнят, ИИ-агент рассуждает |

Каждая ячейка содержит однородные сущности — функции (что делается), без смеси с носителями и описаниями.

Личность и экзоличность реализуются через мастерства:

| Компонента мастерства | Содержание |
|---|---|
| **Мыслительное мастерство** (интеллект) | Трансдисциплины, способность рассуждать, строить модели |
| **Прикладное мастерство** | Профессиональные методы, навыки конкретных ролей |
| **Эмоциональная регуляция** | Впечатления, убеждения, ценности; механизм перепрограммирования через опыт |

## 3. Модульная декомпозиция

По FPF A.7: модуль = носитель (carrier) + содержание (description). Декомпозиция по типу носителя.

| Модуль | Носитель (carrier) | Содержание (description) |
|---|---|---|
| **Биологическое тело** | Биологические структуры: мышцы, нервная система, мозг как орган, рецепторы | Информация в нейронных связях: мастерства, образы, убеждения, мировоззрение |
| **Экзотело** | Приобретённые материальные артефакты: одежда, автомобиль, ручной инструмент, компьютер, смартфон, бумага | Записанная на носителе информация: программы, ИИ-модели, заметки, тексты, кодовые базы |

Каждая ячейка содержит однородные сущности — либо носители (физические артефакты), либо содержание (информация на этих носителях).

Различение модулей (PD.D.050):
- *Биологическое тело* — дано при рождении, не передаётся другому, эволюционировало биологически
- *Экзотело* — получено технологически: можно купить, продать, подарить; результат техноэволюции

## 4. Связь функция ↔ модуль (мост между декомпозициями)

Каждая функция реализуется на определённом модуле — и носитель, и содержание этого модуля участвуют в реализации:

| Функция (что делает) | Модульная реализация |
|---|---|
| **Организм** (аппаратные функции био) | Биологическое тело — преимущественно носитель (мышцы, рефлексы, метаболизм); содержание участвует минимально (генетические программы) |
| **Личность** (психические функции био) | Биологическое тело — мозг как носитель + информация в нейронных связях как содержание |
| **Экзоорганизм** (аппаратные функции экзо) | Экзотело — преимущественно носитель (физические инструменты, одежда, средства передвижения) |
| **Экзоличность** (психические функции экзо) | Экзотело — компьютер/смартфон как носитель + программы/модели/заметки как содержание. Привычное имя для этой модульной сборки — **экзокортекс** |

«Экзокортекс» в практике саморазвития — устоявшееся имя для модульной сборки (компьютер с программами и моделями), реализующей экзоличность.

## 5. Роль экзокортекса (модульная сборка для экзоличности)

- Вычислительная мощность биологического мозга не изменилась за тысячи лет
- За счёт экзокортекса современный человек имеет на порядок более высокие психические возможности (экзоличность)
- Не все люди умеют использовать экзокортекс — необходимо осваивать практики саморазвития
- Современный человек отличается от человека прошлого именно за счёт развитой экзоличности

## 6. Впечатления как механизм обучения

При действии в роли агент получает два результата:
1. **Рабочий продукт** — изменение в мире
2. **Впечатление** — след, отпечатывающийся в содержании биологического тела (мозг → нейронные связи); изменяет мировоззрение, формирует убеждения, может дать новое мастерство при успехе

## 7. Следствия для саморазвития

Для полноценного саморазвития необходимо уделять внимание всем четырём функциональным категориям (с учётом разных модульных средств их реализации):

| Функция | Метод развития |
|---|---|
| **Организм** (аппаратные функции био) | Сон, питание, движение, восстановление, ЗОЖ — методы для биологического носителя |
| **Личность** (психические функции био) | Обучение, тренировка мастерств, рефлексия, мышление письмом — методы для содержания биологического носителя |
| **Экзоорганизм** (аппаратные функции экзо) | Заработать, купить, обустроить — методы для приобретения и поддержания материальных артефактов |
| **Экзоличность** через экзокортекс (психические функции экзо) | Настроить, описать, обучить ИИ-агента, освоить инструмент — методы для информационного содержания на материальных носителях |

**Не путать функциональные категории с модульными средствами их реализации** (FPF A.7). Один из частых перекосов: пытаются «улучшить экзокортекс деньгами» (купить ещё одну подписку) — это работает на уровне носителя, тогда как реальная функция (экзоличность) растёт через настройку, описание процессов, освоение инструмента.
