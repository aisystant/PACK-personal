# Формализация: 3 контекста стиля жизни

## 1. Назначение

Развитие происходит не только во время учёбы. Человек действует в **трёх контекстах**, и в каждом развиваются оба объекта (мировоззрение + мастерство, PD.FORM.001 §5.1).

**Зачем:** Портной (MIM.SOP.001) в��бирает не только область (FORM.081) и тип воздействия (мировоззрение/мастерство), но и **контекст**, в котором ученик будет практиковаться. Без контекста задание «повисает в воздухе» — непонятно, когда и где его выполнять.

**Что заменяет:** Лестница 29 сту��еней (пост «Лестница методов», 2025-07) **полностью растворяется** в 3 контекстах. Каждая из 29 ступеней — это конкретная практика внутри одного из контекстов. Отдельный артефакт «лестница» не нужен.

## 2. Три контекста

| # | Контекст | Что делает человек | Целевое состояние | Методы (source-of-truth) |
|---|----------|-------------------|-------------------|--------------------------|
| 1 | **Саморазвитие** | Создаёт мастерство: учится, читает, пишет, рефлексирует | Новая компетенция (can-do) | 8 практик: METHOD.001–005, METHOD.008–009 + METHOD.007 |
| 2 | **Работа** | Применяет мастерство: создаёт рабочие продукты | Рабочий продукт, вклад | 14 элементов культуры работы (DP.M.008) |
| 3 | **Досуг и восстановление** | Восстанавливает ресурсы + получает впечатления | Восстановленный организм + воспоминания | METHOD.006 + 10 практик досуга (PD.CAT.002) |

**Знания** (программа ЛР, руководства, Pack) — не четвёртый контекст, а **содержание**, которое подаётся в контексте Саморазвития и компилируется через практику во всех трёх контекстах.

## 3. Граф зависимостей

```
Досуг и восстановление
  │  (без ресурсов нет энергии на учёбу)
  ▼
Саморазв��тие
  │  (без мастерства нечего применя��ь)
  ▼
Работа
  │  (работа порождает неудовлетворённости)
  └──▶ Стратегирование ──▶ новый цикл
```

**Следствия:**
- Засор в Досуге → срывы в Саморазвитии → деградация Работы
- Засор в Саморазвитии → стагнация в Работе (делает одно и то же)
- Засор в Работе → нет результатов → мотивация падает → цикл останавливается

**Диагностика засоров** (источник: пост «Три потока», systemsworld.club/31419):

| Симптом | Засор в |
|---------|---------|
| Много начато, мало результатов | Работа |
| Читаю много, голова расплывчата, не могу объяснить | Саморазвитие |
| Не помню неделю, жизнь скучна, нет впечатлений | Досуг (впечатления) |
| Срывы внимания, туман, усталость без причины | Досуг (энергия) |

## 4. Контекст 1: Саморазвитие

**Что внутри:** Создание мастерства через 8 практик ученика.

| Слой | Практики | Когда запускать |
|------|----------|----------------|
| 0 (фундамент) | Учёт времени (METHOD.001) + Стр��тегирование (METHOD.008) | Ступень 1 |
| 1 (инфраструктура) | Планирование (METHOD.009) + Организация досуга (METHOD.006) | Ступень 1–2 |
| 2 (конвейер) | Систематическое чтение (METHOD.003) → Мышление письмом (METHOD.004) → Мышление проговариванием (METHOD.005) | Ступень 2–3 |
| 3 (среда) | Формирование окружения (METHOD.007) | Ступень 3–4 |

**Порядок слоёв — не произвольный.** Каждый следующий слой опирается на предыдущий:
- Без учёта времени (слой 0) непонятно, куда уходит время → планирование (слой 1) невозможно
- Без чтения (слой 2) нечего писать → мышление письмом буксует
- Без письма нечего проговаривать → проговаривание поверхностно

**Связь с FORM.081 (5 областей):** Саморазвитие затрагивает преимущественно области «Знания» и «Инструменты», но не исключительно — стратегирование работает с «Ограничениями», формирование окружения — с «Окружением».

## 5. Контек��т 2: Работа

**Что внутри:** Применение мастерства через культуру работы.

14 элементов культуры работы IWE (DP.M.008): ОРЗ, каскад, учёт, Capture, АрхГейт, экзоскелетный режим и др.

**Ключевое:** Работа — не «отдельное от развития». Работа = контекст, в котором мастерство проверяется. Без Работы мастерство остаётся теоретическим (FAIL.055: знания без компиляции).

**Для ученика ступеней 0–2:** Работа = учебные задания, мини-проекты, рабочие продукты по руководствам.
**Для ступеней 3–4:** Работа = реальные проекты, привязанные к неудовлетворённостям.

## 6. Контекст 3: Досуг и восстановление

**Что внутри:** 2 подпотока.

| Подпоток | Практики | Назначение |
|----------|----------|------------|
| **Энергия** | Сон, питание, саморегуляция, движение, тишина и отключение, медчек | Восстановление ресурсной базы |
| **Впечатления** | Замена удовольствий, микро-приключения, путешествия и смена контекста, фиксация впечатлений | Наполнение жизни смыслом и воспоминаниями |

**Управление:** METHOD.006 (организация досуга) — практика из группы 1 (Саморазвитие), но объект управления — контекст 3 (Досуг). Эшелонирование = принцип управления: практики досуга на всех временных масштабах (час, день, неделя, месяц, год).

**Каталог практик досуга:** PD.CAT.002 (10 практик с can-do, диагностикой засоров, привязкой к ступеням).

## 7. Привязка к ступеням ученика (FORM.003)

| Ступень | Ведущий контекст | Что происх��дит |
|---------|-----------------|----------------|
| 0 → 1 (Случайный → Практикующий) | **Досуг** + Саморазвитие | Стабилизация ресурсов (сон, режим) + первые практики (учёт времени, чтение) |
| 1 → 2 (Практикующий → Систематический) | **Саморазвитие** | Конвейер запущен: чтение → письмо → планирование. Досуг поддерживает |
| 2 → 3 (Систематический → Дисциплинированный) | **Саморазвитие** + Работа | Стратегирование связывает учёбу с проектами. Работа = контекст проверки |
| 3 → 4 (Д��сциплинированный → Проактивный) | **Работа** + Досуг | Работа с реальными проектами. Досуг = стиль жизни (не выживание) |

**Принцип:** На ранних ступенях фокус на Досуге (стабилизация) и Саморазвитии (конвейер). На поздних — на Работе (результаты) и зрелом Досуге (впечатления, а не только энергия).

## 8. Растворение лестницы 29 ступеней

Лестница методов (пост 2025-07) содержала 29 поддерживающих практик, сгруппированных по «ступеням» (от сна до медчека). Каждая из 29 ступеней — это конкретная практика внутри одного из 3 контекстов:

| Бывшая группа лестницы | Контекст | Куда перешло |
|------------------------|----------|-------------|
| Сон, питание, движение, медчек | Досуг → Энергия | PD.CAT.002 |
| Замена удовольствий, микро-приключения | Досуг �� Впечатления | PD.CAT.002 |
| Учёт времени, чтение, письмо, планирование | Саморазвитие | METHOD.001–005, METHOD.008–009 |
| Стратегирование, формирование окружения | Саморазвитие (слой 3) | METHOD.007–008 |
| ОРЗ, каскад, Capture | Работа | DP.M.008 |

**Отдельный артефакт «лестница» не нужен.** Три контекста + каталоги (METHOD, CAT.002, DP.M.008) покрывают всё содержание.

Портной на Шаге 2 выбирает: (1) область (FORM.081), (2) тип (мировоззрение/мастерство), (3) **контекст** (FORM.082) → конкретный элемент из каталога этого контекста.
