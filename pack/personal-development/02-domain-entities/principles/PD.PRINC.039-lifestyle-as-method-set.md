# Принцип: Стиль жизни как инженерная совокупность методов (Lifestyle as Method Set)

## 1. Формулировка

**Стиль жизни — не привычки, а проектируемый набор методов.**

Привычка — автоматизм без рефлексии. Метод — осознанная практика с критериями корректности. Стиль жизни созидателя — это архитектура из взаимосвязанных методов, каждый из которых выбран, а не унаследован.

## 2. Следствие

Изменение стиля жизни — это замена одних методов другими, а не «формирование привычек». Каждый метод имеет описание, рабочие продукты и критерии корректности.

## 3. Индикаторы нарушения

- «Просто нужно привыкнуть»
- Отсутствие явного описания практик, составляющих стиль жизни
- Изменения через силу воли без изменения метода
