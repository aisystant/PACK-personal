# Принцип: Сначала ответь сам, потом спрашивай (Self-Answer First)

## 1. Формулировка

**Проработать вопрос на творческом конвейере до обращения за помощью.**

Вопрос, заданный без предварительной собственной проработки, — это делегирование мышления. Собственная версия ответа (даже ошибочная) создаёт контекст, в котором чужой ответ усваивается, а не просто потребляется.

## 2. Следствие

Обращение за помощью содержит собственную версию ответа и описание того, где именно проработка остановилась. Качество вопроса определяется глубиной предшествующей самостоятельной работы.

## 3. Индикаторы нарушения

- Вопрос без собственной версии ответа
- «Я не знаю» без попытки разобраться
- Немедленное обращение к эксперту при первом затруднении
