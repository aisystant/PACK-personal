# Принцип: Mental Model as Ultimate Constraint

## 1. Формулировка

**В знаниевой работе ментальная модель (устоявшееся убеждение о том «как всё устроено») — наиболее трудноустранимый вид ограничения.**

В manufacturing ограничение — станок или процесс: видно, измеримо, устраняется наймом или заменой оборудования. В knowledge work ограничением чаще служит Mental Model: «у нас так принято», «это невозможно изменить», «всегда делали по-другому». Такой constraint не устраняется ресурсами — требует изменения убеждения.

## 2. Следствие

Перед попыткой Elevation (добавить ресурс, купить инструмент, нанять) — проверить: не является ли ограничение Mental Model или policy (правилом, вытекающим из Mental Model). Если да — Five Focusing Steps неприменимы; нужен Evaporating Cloud (MIM.M.015) для выявления допущения, удерживающего ограничение.

## 3. Иерархия типов constraints (TameFlow, Tendon 2020)

| Тип | Описание | Инструмент |
|-----|----------|-----------|
| **Work Flow** | Самый длинный workload across pipeline | Five Focusing Steps |
| **Work Process** | Самый длинный avg flow time within stage | Five Focusing Steps |
| **Work Execution** | Динамическое in-the-moment ограничение | Immediate action |
| **Policy / Mental Model** | Правило или убеждение, удерживающее constraint | Evaporating Cloud |

## 4. Связь с программой личного развития

Mental Model как constraint релевантен на всех ступенях развития, но проявляется по-разному:

- **Ступень 1–2 (случайный / практикующий)**: Mental Model = «у меня нет времени / способностей / ресурсов» — барьер к началу
- **Ступень 3–4 (систематический / дисциплинированный)**: Mental Model = «системы не меняются» / «это политика, не мой уровень» — барьер к масштабированию влияния
- **Ступень 5 (проактивный)**: Mental Model = «невозможно передать культуру» — барьер к репликации

## 5. Индикаторы наличия

- Bottleneck не устраняется при добавлении ресурсов
- В команде/системе звучат фразы «у нас так принято», «это невозможно», «всегда так делали»
- Constraint мигрирует после каждой попытки устранения (moving Herbie)
