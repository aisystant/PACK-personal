# Состояние: Шкала продуктивных состояний (0–5)

## 1. Определение

Шесть именованных уровней продуктивного состояния, ранжированных от 0 (критическое) до 5 (пиковое). Полное описание каждого уровня (признаки, спектр, механизмы управления через досуг) — [PD.FORM.038](../formalizations/PD.FORM.038-productive-state-scale.md), эта карточка — только каталожная запись + маппинг на допустимые задачи (PD.FORM.068). Шкала дискретная: промежуточные значения (например, 2.5) не используются — агент выбирает ближайший уровень.

> **Сокращено 2026-07-12** (детектор непротиворечивости WP-429 нашёл дословное повторение таблицы уровней из FORM.038). Полное описание уровня — по ссылке выше, здесь — только допустимые задачи и приоритет.

## 2. Маппинг на допустимые задачи

| Уровень | Имя (FORM.038 §2) | Допустимые задачи | Приоритет |
|---------|--------------------|--------------------|-----------|
| **5** | Peak | Всё: архитектура, стратегия, творчество, освоение нового | Важное и когнитивно-нагруженное |
| **4** | Functional | Сложные задачи, глубокое чтение, принятие решений | Важное |
| **3** | Satisfactory | Текучка, коммуникация, знакомые задачи | Текучка и поддержка |
| **2** | Decline | Только простое: сортировка, уборка, административное | Восстановление + простые дела |
| **1** | Failure | Минимум рутины, не требующей концентрации | Восстановление |
| **0** | Critical | СТОП. Только базовый уход: сон, еда, безопасность | Выживание |

## 3. Динамика переходов

```
Peak (5) ←→ Functional (4) ←→ Satisfactory (3) ←→ Decline (2) ←→ Failure (1) ←→ Critical (0)
```

- Подъём: восстановление, сон, физическая активность, положительные события.
- Спад: переутомление, стресс, болезнь, нарушение режима.
- Скорость перехода асимметрична: спад происходит быстрее подъёма.
- Типичный рабочий диапазон: 3–4. Устойчивое пребывание на 5 — редкость; на ≤1 — сигнал тревоги.
