# Состояние: Роле-релевантное состояние (Role-Relevant State)

## 1. Определение

Набор параметров агента, определяющих, может ли агент реально выполнить метод, ожидаемый текущей ролью. Роле-релевантные параметры различаются по ролям.

## 2. Аспекты

| Аспект | Примеры параметров |
|--------|--------------------|
| Физический | Температура тела, восстановление, травмы, качество техники |
| Когнитивный | Фокус, ясность приоритетов |
| Эмоциональный | Стабильность в конфликте |
| Ресурсный | Время, доступ, бюджет |
| Компетентностный | Мастерство метода, навык координации |

## 3. Ключевое наблюдение

Разрыв между ожиданиями роли и роле-релевантным состоянием — **основная причина «ролевого провала»** (не плохие люди и не неправильная оргструктура).

## 4. Диагностика

Для разных ролей — разные наборы параметров:
- **Спортсмен**: температура тела, восстановление, статус травмы
- **Менеджер проекта**: ясность приоритетов, способность к фокусу, мастерство координации, эмоциональная стабильность
