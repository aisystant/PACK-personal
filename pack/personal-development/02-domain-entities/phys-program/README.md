# Phys-program: физический слой ст. 0 (saneness)

> **Pre-condition для входа в программу личного развития** (см. [PD.FORM.080 §4 «Ступень 1: Случайный ученик»](../formalizations/PD.FORM.080-stage-direction-normative-matrix.md)).
> Слой 0 saneness — необходимое условие устойчивого ритма 10ч/нед инвестированного времени ([PD.FORM.097 §4](../formalizations/PD.FORM.097-worldview-as-executable-code.md)).

---

## Назначение каталога

Каталог `phys-program/` фиксирует онтологию физического слоя как **ст. 0 saneness** программы личного развития (ЛР). Это **снаружи** программы (предусловие), не первая ступень.

**Что описано:**
- 4 субслоя физического слоя (сон / движение / питание / восстановление)
- Метрики saneness каждого субслоя (baseline, не оптимум)
- Триггер реактивации программы при просадке субслоя
- Гейт перехода ст. 0 → ст. 1
- Роль, отвечающая за проверку saneness

**Что не описано (вне scope):**
- Методы оздоровления (компетенция тренера/врача/диетолога — downstream)
- Тренировочные планы (cardio, силовая, гибкость)
- Диеты, нутрициология, биохакинг
- Медицинские интервенции, психотерапия

## Failure mode без физического слоя

Без онтологии физического слоя ст. 0 проявляется [PD.FAIL.021 «Ignoring Sleep and Daily Routine»](../../05-failure-modes/PD.FAIL.021-ignoring-sleep-and-daily-routine.md): «переход к продвинутым практикам саморазвития при сломанном сне и отсутствии распорядка дня — главная фундаментальная ошибка». PD.FORM.138 даёт positive-определение того же явления.
