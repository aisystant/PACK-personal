# Шаблоны пакета Lifework

> Шесть шаблонов документов пакета Lifework по уровням охвата созидателя (PD.FORM.090). Реализуют метод [PD.METHOD.018 Lifework Pack Composition](../../03-methods/PD.METHOD.018-lifework-pack-composition.md).

## Принцип использования

Шаблон — структурный каркас документа с вопросами-провокаторами для заполнения и чеклистом готовности. **Не вписывать в шаблон желаемое; вписывать только то, что уже есть в практике созидателя за последние ≥6 месяцев.** Имитация подъёма (см. PD.METHOD.018 Failure Modes) — главный режим отказа метода.

## Опциональные документы (на верхних уровнях)

Возникают при специфических условиях. Шаблоны — расширения базовых:
- **Манифест сообщества** (уровень 4, при наличии оформленного сообщества): расширение шаблона уровня 4
- **Защита Дела** (уровень 6, при конкуренции со слоем-вендором): отдельный документ; шаблон выделяется при появлении второго применения метода
- **Устав экосистемы** (уровень 6, при масштабе ≥ организации): отдельный документ; шаблон — после второго применения

## Условие подъёма между уровнями

Документ уровня N-1 работает ≥6 месяцев. Конкретные решения за это время приняты против инвариантов уровня N-1. Без удовлетворения условия — подъём = имитация (см. PD.METHOD.018 §«Условия подъёма»).
