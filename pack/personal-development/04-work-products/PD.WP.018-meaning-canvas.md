# [PD.WP.018] Meaning Canvas

---

## Definition

**Meaning Canvas** is a one-page structured document capturing the agent's current understanding of their life meaning across a 4x3 matrix: levels (Self, Close circle, Community, World) crossed with three dimensions of meaning (Coherence, Purpose, Significance) from the Martela & Steger tripartite model. Each cell contains a short formula; each row includes a working formula: "for whom -> by what method -> which system we change -> result metric".

The Meaning Canvas is NOT:
- A mission statement (which is abstract and static; the canvas is concrete and revisable)
- A life plan (which prescribes actions in detail; the canvas captures current understanding)
- A values list (which names values without operationalization; the canvas links values to systems and methods)
- A journal entry (which is narrative; the canvas is structured and inspectable)
- A finished answer (meaning is a process, not a destination; the canvas is a snapshot)

---

## Purpose

| Function | Description |
|----------|-------------|
| **Making invisible visible** | Externalizes the agent's implicit sense of meaning into inspectable form |
| **Gap detection** | Empty cells reveal dimensions where meaning is underdeveloped |
| **Strategic alignment** | Connects existential sense of meaning with concrete projects and systems |
| **Anti-crisis tool** | When meaning feels lost, the canvas shows which dimension collapsed |
| **Strategizing input** | Provides the "why" layer for prioritization of projects and roles |

## Existence Criteria

| Criterion | Test |
|-----------|------|
| **Written document** | Canvas exists as a file, not "in the head" |
| **Matrix filled** | At least 6 of 12 cells contain short formulas (not blank) |
| **Working formula** | At least 2 rows have "for whom -> method -> system -> metric" |
| **Three dimensions** | Coherence, Purpose, and Significance columns are all present |
| **Dated** | Creation date and last update date recorded |

**Existence check**: Can the agent point to one cell in the matrix and explain how it connects to a current project or role?

---

## Quality Criteria

| Criterion | Description | How to Verify |
|-----------|-------------|---------------|
| **Concrete** | Cells contain specific formulas, not platitudes ("make the world better") | Each cell references a role, project, or system |
| **Covers multiple levels** | Not only "Self" row filled; at least 2 levels populated | Inspect matrix |
| **Action-linked** | Working formulas connect to actual projects in Priority Projects List | Cross-reference with PD.WP.007 |
| **Revisable** | Canvas is updated at strategizing (at least quarterly) | Compare timestamps |
| **Honest about gaps** | Empty cells are acknowledged, not hidden | Blank cells visible, not filled with noise |
| **Coherent** | Rows do not contradict each other | Self-level purpose aligns with world-level significance |

## Anti-Patterns

| Anti-Pattern | Why Problematic |
|--------------|-----------------|
| Abstract platitudes | "Help humanity" without method, system, or metric -- untestable |
| Only Self level | Ignoring community and world levels leads to narcissistic meaning |
| Static document | Never revised; meaning evolves, canvas must evolve with it |
| Borrowed meaning | Filling canvas with socially expected answers instead of personal truth |
| Confusing with life plan | Canvas captures understanding of meaning, not a schedule of actions |
| All cells filled at first attempt | Premature completeness signals superficiality; gaps are honest |
