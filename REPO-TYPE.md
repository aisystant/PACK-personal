# Тип репозитория

**Тип**: `Pack`

**Source-of-truth**: yes

## Область

**Характеристики и состояния созидателя**

Инженерное описание созидателя как объекта оценки: характеристики (агентность, калибр личности), состояния (ученика), методы оценки, рабочие продукты, типовые ошибки интерпретации.

## Upstream dependencies

- [TserenTserenov/SPF](https://github.com/TserenTserenov/SPF) — Second Principles Framework
- [ailev/FPF](https://github.com/ailev/FPF) — First Principles Framework

## Downstream outputs

- [aisystant/DS-twin](https://github.com/aisystant/DS-twin) — использует индикаторы из этого pack
- [aisystant/DS-aist-bot](https://github.com/aisystant/DS-aist-bot) — реализует марафон на основе этого pack
- Курсы и руководства по личному развитию (surface)

## Non-goals

- НЕ содержит курсов, уроков, маршрутов обучения (это downstream)
- НЕ содержит кода, сервисов, MCP (это downstream-instrument)
- НЕ содержит управленческих процессов (это downstream-governance)
- НЕ описывает «как развить характеристику» — только «что это и как измерить»
